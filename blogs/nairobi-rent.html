Nairobi Rental Price Prediction

Machine Learning · Regression

Rental prices in Nairobi vary widely depending on location, property size, and available amenities. This project applies machine learning techniques to predict monthly rental prices based on structured property data.

Problem Statement

The goal of this project was to build a predictive model that estimates monthly rent for residential properties in Nairobi using key features such as location, number of bedrooms, bathrooms, parking availability, estimated size, and distance to the Central Business District (CBD).

Dataset & Features

The dataset consisted of structured housing records collected from rental listings. Key features used in the model included:

  • Number of bedrooms and bathrooms
  • Parking availability
  • Estimated property size
  • Location / neighborhood
  • Distance to Nairobi CBD

Methodology

The workflow followed a standard machine learning pipeline:

  • Data cleaning and preprocessing
  • Exploratory Data Analysis (EDA)
  • Feature encoding and scaling
  • Training regression models
  • Model evaluation and comparison

Multiple regression algorithms were tested, and performance was evaluated using error metrics such as Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE).

Key Insights

  • Location had the strongest influence on rental prices
  • Properties closer to the CBD were significantly more expensive
  • Size and number of bedrooms showed strong positive correlation with rent

Conclusion

This project demonstrates how machine learning can support data‑driven decision‑making in real estate. The final model provides reasonable rental price estimates and can be extended with additional data such as property age or market trends.
